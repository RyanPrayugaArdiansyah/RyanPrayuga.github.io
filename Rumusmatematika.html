Rumus Matematika

Rumus Kecepatan, Waktu, dan Jarak

Menghitung kecepatan : V = S/t
Menghitung Waktu : t = S/V
Menghitung Jarak : S = t x V

Keterangan
V : kecepatan (km/jam)
S : jarak (km)
t : waktu (jam)

Rumus Bangun Datar

a. Persegi

Rumus luas persegi (sisi dikali sisi) : L = s x s
Rumus keliling persegi (4s), atau : K = 4 × s

b. Persegi Panjang

Rumus luas persegi panjang (panjang X lebar) : L = p × l
Rumus keliling persegi (4s) atau : K = 2 × (p + l)

c. Segitiga

Rumus luas segitiga (alas kali tinggi dibagi dua) : L = ½ × a × t
Rumus keliling segitiga (menjumlahkan semua sisi) : K = a + b + c

d. Trapesium

Rumus luas trapesium : L = ½ × (a + b) × t
Rumus keliling trapesium : K = AB + BC + CD + DA

e. Jajar Genjang

Rumus luas Jajar Genjang : L = a × t
Rumus keliling Jajar Genjang : K = 2 × (a + b)

f. Layang-layang

Rumus luas layang-layang : L = ½ x diagonal (d) 1 x diagonal (d) 2
Adapun cara menghitung diagonal adalah :
Diagonal 1 (d1) = 2 × L ÷ d2
Diagonal 2 (d2) = d2 = 2 × L ÷ d1
Rumus keliling layang-layang : = a + b + c + d atau Kll = 2 × (a + c)

g. Belah Ketupat

Rumus luas belah ketupat : L = ½ x diagonal (d) 1 x diagonal (d) 2
Rumus keliling belah ketupat : K = s + s + s + s atau s × 4

h. Lingkaran

Rumus luas lingkaran : L = π (pi) x jari-jari (r) kuadrat
Rumus keliling lingkaran : K = π × diameter atau π × r2

Rumus Bangun Ruang

a. Kubus

Rumus Volume Kubus : V= s x s x s

b. Balok

Rumus Volume Balok : V = p x l x t

c. Prisma

Rumus volume prisma : V = Luas segitiga x tinggi

d. Limas

Rumus volume limas : V = 1/3 Luas Alas x Tinggi

e. Kerucut

Rumus volume kerucut : V = 1/3 x π x r x r x t

f. Tabung

Rumus volume tabung : V = π x r2 x t

g. Bola

Rumus volume bola : V = 4/3 x π x r3
